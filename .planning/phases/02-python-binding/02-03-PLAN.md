Transform MessageIterator from sync iterator to async iterator:

1. Keep __iter__/__next__ for backwards compatibility (sync iteration still works)

2. Add async iterator protocol:
   ```rust
   fn __aiter__(slf: PyRef<'_, Self>) -> PyRef<'_, Self> {
       slf
   }

   fn __anext__<'py>(&self, py: Python<'py>) -> PyResult<Option<Bound<'py, PyAny>>> {
       let rx = self.receiver.clone();
       let timeout = self.timeout;

       future_into_py(py, async move {
           // Receive with timeout to allow cancellation
           let result = match timeout {
               Some(t) => tokio::time::timeout(t, rx.recv()).await,
               None => Ok(rx.recv().await),
           };

           match result {
               Ok(Some(msg)) => {
                   Python::with_gil(|py| {
                       let dict = message_to_dict(py, &msg)?;
                       Ok(Some(dict.into_py(py)))
                   })
               }
               Ok(None) => Ok(None), // Channel closed -> StopAsyncIteration
               Err(_) => Ok(None),   // Timeout -> return None, don't stop
           }
       })
   }
   ```

3. Use tokio's async mpsc channel instead of std::sync::mpsc:
   - Change receiver type to Arc<tokio::sync::mpsc::UnboundedReceiver<WebSocketMessage>>
   - This enables async recv() without blocking

4. Add try_recv_nowait() for non-blocking check (sync, for polling use cases)

CRITICAL GIL safety:
- Never hold GIL while awaiting
- Use Python::with_gil() only for final result conversion
- Use allow_threads() if any sync blocking is needed

   ```rust
   pub fn connect<'py>(&self, py: Python<'py>) -> PyResult<Bound<'py, PyAny>> {
       let client = self.inner.clone();
       let callbacks = self.callbacks.clone();

       future_into_py(py, async move {
           client.connect().await
               .map_err(|e| crate::errors::to_py_err(e))?;

           // Emit connect event via callback
           Python::with_gil(|py| {
               callbacks.invoke_connect(py);
           });

           Ok(())
       })
   }
   ```

2. Convert subscribe() to async and return subscription handle:
   ```rust
   pub fn subscribe<'py>(&self, py: Python<'py>, channel: String, symbol: String) -> PyResult<Bound<'py, PyAny>> {

       future_into_py(py, async move {
           let sub_id = client.subscribe(&channel, &symbol).await
               .map_err(|e| crate::errors::to_py_err(e))?;
           Ok(sub_id)
       })
   }
   ```
