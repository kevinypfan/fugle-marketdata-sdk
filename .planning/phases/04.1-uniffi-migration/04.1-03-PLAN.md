3. **Message forwarding task:**
Spawn a tokio task that:
- Receives messages from core WebSocket's MessageReceiver
- Converts to StreamMessage
- Calls listener.on_message()

```rust
async fn run_message_loop(
    mut receiver: MessageReceiver,
    listener: Arc<dyn WebSocketListener>,
) {
    while let Some(msg) = receiver.recv().await {
        let stream_msg = StreamMessage::from(msg);
        listener.on_message(stream_msg);
    }
    listener.on_disconnected();
}
```
