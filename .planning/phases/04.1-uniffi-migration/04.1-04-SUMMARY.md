## Key Decisions

| Decision | Rationale |
|----------|-----------|
| Library-mode generation | Project uses proc-macro approach (uniffi::setup_scaffolding!) not UDL files |
| Post-process to public | Generated types are `internal`; sed post-processing makes them `public` for consumer access |
| Wrapper in same project | Avoids InternalsVisibleTo complexity; single assembly for consumers |
| Return internal types | Wrapper methods return `uniffi.marketdata_uniffi.Quote` directly (not re-exported types) |
| Multi-target framework | netstandard2.0 for broad compat, net6.0/net8.0 for modern features |

## Deviations from Plan

### [Rule 3 - Blocking] Adapted for proc-macro approach
- **Found during:** Task 1
- **Issue:** Plan referenced UDL file but STATE.md explicitly notes project uses proc-macro approach
- **Fix:** Used `uniffi-bindgen-cs --library` mode instead of UDL mode
- **Impact:** None - library mode works correctly with proc-macro scaffolding

### [Rule 2 - Missing Critical] Post-processing for public visibility
- **Found during:** Task 3
- **Issue:** Generated types are `internal`, causing CS0050 errors when exposed via public wrapper
- **Fix:** Added sed post-processing in Makefile to change `internal` to `public`
- **Impact:** Minor - standard pattern for generated bindings

### [Rule 1 - Bug] Removed separate wrapper project
- **Found during:** Task 3
- **Issue:** Separate FugleMarketData project couldn't access internal types even with InternalsVisibleTo
- **Fix:** Moved wrapper classes into MarketdataUniffi project
- **Impact:** Simpler single-assembly design; consumers use `FugleMarketData` namespace

## Usage Example

```csharp
using FugleMarketData;
using uniffi.marketdata_uniffi;

// Create client
using var client = new RestClient("your-api-key");

// Async usage (preferred)
var quote = await client.Stock.Intraday.GetQuoteAsync("2330");
Console.WriteLine($"TSMC: {quote.lastPrice}");

// Sync usage
var ticker = client.Stock.Intraday.GetTicker("2330");
Console.WriteLine($"Name: {ticker.name}");

// WebSocket streaming
class MyListener : IWebSocketListener
{
    public void OnConnected() => Console.WriteLine("Connected!");
    public void OnDisconnected() => Console.WriteLine("Disconnected");
    public void OnMessage(StreamMessage msg) => Console.WriteLine($"{msg.channel}: {msg.symbol}");
    public void OnError(string error) => Console.WriteLine($"Error: {error}");
}

using var ws = new WebSocketClient("your-api-key", new MyListener());
await ws.ConnectAsync();
await ws.SubscribeAsync("trades", "2330");
```
