2. **StockIntradayClient with async methods:**
```rust
pub struct StockIntradayClient {
    inner: marketdata_core::RestClient,
}

#[uniffi::export(async_runtime = "tokio")]
impl StockIntradayClient {
    pub async fn get_quote(&self, symbol: String) -> Result<Quote, MarketDataError> {
        let core_quote = tokio::task::spawn_blocking({
            let inner = self.inner.clone();
            let symbol = symbol.clone();
            move || inner.stock().intraday().quote().symbol(&symbol).send()
        })
        .await
        .map_err(|e| MarketDataError::RuntimeError(e.to_string()))??;

        Ok(core_quote.into())  // Convert using From trait from models.rs
    }

    pub async fn get_ticker(&self, symbol: String) -> Result<Ticker, MarketDataError> {
        // Same pattern
    }

    // Sync variants (blocking wrapper)
    pub fn quote_sync(&self, symbol: String) -> Result<Quote, MarketDataError> {
        let rt = tokio::runtime::Handle::current();
        rt.block_on(self.get_quote(symbol))
    }
}
```

3. **Use spawn_blocking** because core REST client uses blocking ureq (per Phase 2 decisions).

4. **Convert all responses** using Into<UniffiModel> from models.rs From implementations.

Note: If core uses blocking HTTP, wrap in spawn_blocking. If core has async, use directly.
