# Phase 04.1 Plan 03: WebSocket Client with Foreign Trait Callbacks

WebSocket client implementation with `#[uniffi::export(with_foreign)]` trait pattern for typed streaming data delivery to C#/Go consumers.

## One-liner

WebSocket client with WebSocketListener foreign trait using spawn_blocking bridge for MessageReceiver timeout polling.

## What Was Built

### WebSocket Client Architecture

```
Foreign Code (C#/Go)                 Rust (UniFFI)
+---------------------+              +---------------------+
| class MyListener    |              | WebSocketClient     |
|   implements        |--callback----+   spawns message    |
|   IWebSocketListener|              |   forwarding task   |
|                     |              |                     |
| OnMessage(msg) <----+--------------+ run_message_loop()  |
| OnConnected()  <----+              |                     |
| OnDisconnected()<---+              | CoreWebSocketClient |
| OnError(err)   <----+              |   .messages()       |
+---------------------+              +---------------------+
```

### Components Implemented

1. **WebSocketListener trait** (`#[uniffi::export(with_foreign)]`)
   - `on_connected()` - Connection established callback
   - `on_disconnected()` - Connection closed callback
   - `on_message(StreamMessage)` - Typed message delivery
   - `on_error(String)` - Error notification

2. **WebSocketClient** (`#[derive(uniffi::Object)]`)
   - Constructors: `new()`, `new_with_endpoint()`
   - Async methods: `connect()`, `subscribe()`, `unsubscribe()`, `disconnect()`
   - Sync method: `is_connected()`

3. **WebSocketEndpoint enum**
   - `Stock` - Stock market data endpoint
   - `FutOpt` - Futures/options endpoint

4. **StreamMessage conversion**
   - `From<core::WebSocketMessage>` implementation
   - Extracts error_code/error_message from error events

### Key Implementation Details

**Foreign Trait Pattern:**
```rust
#[uniffi::export(with_foreign)]
pub trait WebSocketListener: Send + Sync {
    fn on_connected(&self);
    fn on_disconnected(&self);
    fn on_message(&self, message: StreamMessage);
    fn on_error(&self, error_message: String);
}
```

**Message Loop with spawn_blocking:**
```rust
async fn run_message_loop(...) {
    loop {
        if shutdown.load(Ordering::SeqCst) { break; }

        let result = tokio::task::spawn_blocking(move || {
            receiver_clone.receive_timeout(Duration::from_millis(100))
        }).await;

        match result {
            Ok(Ok(Some(ws_msg))) => {
                listener.on_message(StreamMessage::from(ws_msg));
            }
            // ... handle timeout, errors
        }
    }
}
```
