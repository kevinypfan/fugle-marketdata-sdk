  <name>Task 1: Create Python and JavaScript migration scripts</name>
  <files>migration/migrate-python.py, migration/migrate-javascript.js</files>
  <action>
Create migration/ directory at project root.

Create migration/migrate-python.py:
- Use libCST (not lib2to3 or regex) for lossless CST transformation
- Import libcst, libcst.matchers, argparse, pathlib
- Implement a CSTTransformer class that handles:
  1. `RestClient("key")` -> `RestClient(api_key="key")` (single positional string -> api_key kwarg)
  2. `WebSocketClient("key")` -> `WebSocketClient(api_key="key")` (same pattern)
  3. `RestClient(variable)` -> `RestClient(api_key=variable)` (positional variable -> api_key kwarg)
  4. `RestClient.with_bearer_token("token")` -> `RestClient(bearer_token="token")` (static method -> kwarg)
  5. `RestClient.with_sdk_token("token")` -> `RestClient(sdk_token="token")` (static method -> kwarg)
- ONLY transform calls where the function matches Name("RestClient") or Name("WebSocketClient") or Attribute matching .with_bearer_token/.with_sdk_token
- Skip if call already uses keyword arguments (already migrated)
- CLI interface with argparse:
  - `--path` (required): directory or file to transform
  - `--dry-run`: print diff without writing changes
  - `--verbose`: show each file processed
- Process .py files recursively in given path
- Print summary: N files processed, M files modified
- Add shebang `#!/usr/bin/env python3` and make executable description
- Include docstring with usage examples:
  ```
  # Dry run (preview changes):
  python migration/migrate-python.py --path src/ --dry-run

  # Apply changes:
  python migration/migrate-python.py --path src/
  ```

Create migration/migrate-javascript.js:
- This is a jscodeshift transform module (not a standalone script)
- Export a single function: `module.exports = function(fileInfo, api, options) { ... }`
- Use api.jscodeshift to transform:
  1. `new RestClient('key')` -> `new RestClient({ apiKey: 'key' })` (Literal string to object)
  2. `new WebSocketClient('key')` -> `new WebSocketClient({ apiKey: 'key' })` (same pattern)
  3. `new RestClient(variable)` -> `new RestClient({ apiKey: variable })` (Identifier to object)
- Match NewExpression where callee.name is 'RestClient' or 'WebSocketClient'
- Only transform if arguments.length === 1 and first arg is a Literal (string) or Identifier
- Skip if first argument is already an ObjectExpression (already migrated)
- Return null if no modifications made (jscodeshift convention for unmodified files)
- Use { quote: 'single' } for toSource to preserve single quotes
- Include header comment with usage:
  ```
  // Usage:
  // npx jscodeshift -t migration/migrate-javascript.js src/
  // npx jscodeshift -t migration/migrate-javascript.js src/ --dry
  ```
- Handle both require() and import patterns (jscodeshift handles this at file level)
  </action>
