  <action>
Create a reusable workflow (workflow_call) for building Python wheels using PyO3/maturin-action.

Platform matrix:
- Linux x86_64: ubuntu-latest, target x86_64, manylinux '2_17'
- Linux aarch64: ubuntu-latest, target aarch64, manylinux '2_17'
- macOS universal2: macos-latest, target universal2
- Windows x64: windows-latest, target x64

Workflow inputs:
- version (string): Version to build (for artifact naming)

Steps per matrix job:
1. actions/checkout@v4
2. PyO3/maturin-action@v1 with:
   - target: ${{ matrix.target }}
   - manylinux: ${{ matrix.manylinux || 'auto' }}
   - args: --release -m py/Cargo.toml --out dist
3. actions/upload-artifact@v4:
   - name: wheels-${{ matrix.os }}-${{ matrix.target }}
   - path: dist/*.whl

Use Swatinem/rust-cache@v2 for faster builds (shared-key: python-release).

Important: For aarch64 Linux, maturin-action handles cross-compilation automatically via QEMU. No special setup needed.
  </action>
